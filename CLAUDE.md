# みえるノート - プロジェクト概要

## プロジェクトの目的
副業として、ICL/レーシック領域のアフィリエイトサイトを運営し、月10万円の収益を目指す。

## 背景・戦略
- オーナーは自由診療（美容医療）のマーケティングコンサルティング会社で取締役を務めるデータエンジニア/アナリティクスエンジニア
- 本業は美容医療が中心のため、**美容医療ドメインは避ける**（競業避止リスク）
- ICL/レーシック領域を選定。オーナー自身がICL手術を受ける予定があり、リアルな体験談を書ける
- 将来的にAGA領域も2サイト目として展開予定
- 自由診療アフィリエイトはCPA（成果報酬）が高い（1件1万〜5万円）ため、月2〜10件の成約で目標達成可能

## 制約条件
- 1日に使える時間: 約1時間
- 目標期間: 3ヶ月で成果
- 初期投資予算: 数十万円程度
- 本業との競合を避ける（美容医療は扱わない）

## 技術スタック
- **フレームワーク**: Astro（静的サイト生成）
- **スタイリング**: Tailwind CSS
- **コンテンツ管理**: Astro Content Collections（Markdown）
- **ホスティング**: Vercel（オーナーが使用経験あり）
- **ドメイン**: mieru-note.jp（取得済み）

## サイト設計
- **サイト名**: みえるノート
- **トーン**: 体験記 × 情報メディアの中間（E-E-A-T重視）
- **ターゲットキーワード**: ICL手術、ICL費用、ICL体験談、ICLレーシック比較、ICLクリニック選び

### コンテンツ構成
```
トップページ
├── ICL手術とは？（基礎知識 → what-is-icl.md）
├── ICLの費用相場（検索ボリューム大 → icl-cost.md）
├── ICL vs レーシック比較（CVR高い → icl-vs-lasik.md）
├── ICL体験記（E-E-A-T → icl-experience.md ※随時更新）
├── おすすめクリニック比較（収益記事 → 未作成）
└── 地域別記事（「東京 ICL おすすめ」等 → 未作成）
```

## 収益化の方針
- ASP登録: A8.net、もしもアフィリエイト、アクセストレード等
- 高CPA案件を中心にアフィリエイトリンクを設置
- CTAボックスコンポーネント（CTABox.astro）を記事末尾に配置
- rel="nofollow sponsored" を適切に設定済み

## 集客戦略
1. **SEO**: 特化型コンテンツで検索流入を狙う（3〜6ヶ月で効果）
2. **X（Twitter）**: ICL/視力矯正の情報発信アカウントを開設し、サイトへの導線とする（短期流入）
3. **体験記の随時更新**: 術前→術後の経過記録でリピーターを獲得
